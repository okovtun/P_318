VCS - Version Control System
SVN, Subversion, Mercury,.....
Git != GitHub
Git - система контроля версий
GitHub - облачное хранилище

Repository - это хранилище исходных кодов.
fetch, pull и push
fetch - скачивает изменения с удаленного репозитория на локальный;
pull - скачивает и распаковывает изменения с удаленного репозитория на локальный;
push - загружает изменения с локального репозитория на удаленный;

commit - это контрольная точка, которая содержит снимок репозитория;
commit делают когда хотят сохранить результат работы.
Для того чтобы создать commit, обязательно нужно написать commit-message.
В commit-message программист описывает какие изменения были внесены, 
и какие результаты были достигнуты,
т.е., что именно он хочет сохранить в данный commit.
